# 都立産技高専 TeamЯTR 電装プログラム

鳥人間コンテスト2026 滑空機部門にて使用するプログラムのリポジトリ.

### ログデコーダーの使用方法

SDカードに保存されたバイナリログファイル（`.log`）を、人間が読めるCSV形式に変換するためのスクリプトが `tools/decode_log.py` にあります。
このスクリプトは、タイムスタンプおよびエラーチェック（CRC16）を検証し、センサーIDごとに別々のCSVファイルに復元・出力します。

#### 使用方法
PCにPythonがインストールされている必要があります（追加のパッケージ等は不要です）。

```bash
python tools/decode_log.py <path_to_log_file>
```

実行すると、ログファイルと同じディレクトリに `log_XXXX_MAIN_IMU.csv` や `log_XXXX_GPS.csv` などのCSVファイルが自動的に生成されます。
